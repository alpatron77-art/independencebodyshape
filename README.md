## Publish on GitHub Pages
1. Create a GitHub repo (e.g., `independencebodyshape`)
2. Upload the contents of this folder to the repo root
3. GitHub → Settings → Pages → Deploy from branch → `main` / `(root)`

## Custom domain
Use GitHub Pages settings to set custom domain: `www.IndependenceBodyShape.com`
Then add a CNAME record at your domain host:
- `www` → `<your-github-username>.github.io`
(And redirect the root domain to `www`)
